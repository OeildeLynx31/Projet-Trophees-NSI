# Crédits et Ressources du Projet

Ce document répertorie toutes les ressources tierces utilisées dans le projet **Projet-Trophees-NSI**
Chacune des ressources présentes dans le jeu, sons et images inclus, sont le résultat de création originales, faites par l'Equipe.

## Logiciels et Bibliothèques
- **Langage** : [Python 3.12+](https://www.python.org/)
- **Moteur de jeu** : [Pygame 2.6.1](https://www.pygame.org/)
- **Traitement d'images** : [Pillow (PIL)](https://python-pillow.org/)

## Ressources Graphiques (Assets)
- **Polices de caractères** :
  - *Alagard*
  - *Yoster Island*
- **Sprites** :
  - Personnages et Monstres : Créations originales
  - Interface Utilisateur (UI) : Pack d'interface pixel-art (boutons, barres de vie, slots d'inventaire)

## Ressources Sonores
- **Musiques** :
  - `full_music.mp3` : Musique de jeu
  - `powerful_adventure.mp3` : Musique d'ambiance (inspiration originale d'Aria Math de C418)
  - `base_loop.mp3` : Boucle sonore thématique

## Utilisation de l'IA et de sources tierces

Le projet se veut 100% sans IA, ainsi, aucune source tiers n'a été utilisée pour la réalisation du projet, à l'exception d'une inspiration pour le design du fond d'écran de la page principale que nous avons refait à la main.


## Inspirations du projet

- Hollow Knight (combat)
- Celeste (parcours)
- Clair Obscur: Expédition 33 (univers)
